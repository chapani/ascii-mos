# ASCII-MOS: Amaliy Misollar

Ushbu sahifada joriy o‘zbek lotin yozuvi va **ASCII-MOS** o‘rtasidagi farqlar dasturlov misolida ko‘rsatilgan.

### 1. Ma'no to‘qnashuvlari

| Asl so‘z | Ma'nosi | Standart ASCII | ASCII-MOS |
| :--- | :--- | :--- | :--- |
| **O‘gir** | Convert | ogir | oegir |
| **Og‘ir** | Heavy | ogir | oghir |
| **Bo‘sh** | Empty | bosh | boesh |
| **O‘g‘ri** | Thief | ogri | oeghri |


### 2. Dasturlovda nomlash namunasi

Quyida <abbr title="funksiya">topshiriq</abbr> yoki <abbr title="o‘zgaruvchi">yorliq</abbr> kabi
dasturlov qismlarini nomlashda ASCII-MOS'dan foydalanish namunasi keltirilgan:

```javascript
// Standart yondashuvda (chalkash):
function ogir() { ... } // Bu o‘girmi yoki og‘irmi?

// ASCII-MOS yondashuvida (aniq):
function oegir() { ... }
const oghirYuk = 100;
const boeshjoy = true;
```
