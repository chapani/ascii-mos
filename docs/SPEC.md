# ASCII-MOS: Texnik Tavsif

Ushbu hujjat ASCII-MOS standartining harfiy oʻzgarishlarini belgilab beradi.

### Asosiy Qoidalar


| Harf | ASCII-MOS Koʻrinishi | Mantiq |
| :--- | :--- | :--- |
| **Oʻ / oʻ** | `oe` | Qoʻsh-harf mantigʻi (ö -> oe) |
| **Gʻ / gʻ** | `gh` | Fonetik standart |
| **’** (tutuq) | (tashlab ketiladi) | Qisqalik uchun |

### Katta va Kichik Harflar

1. **Kichik:** `oʻzgarish` -> `oezgarish`, `gʻalvir` -> `ghalvir`
2. **Bosh harf:** `Oʻzgarish` -> `Oezgarish`, `Gʻalvir` -> `Ghalvir`
3. **HAMMASI KATTA:** `OʻZGARISH` -> `OEZGARISH`, `GʻALVIR` -> `GHALVIR`
